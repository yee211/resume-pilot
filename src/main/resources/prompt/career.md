## 候选人简历
{{resume}}

## 目标信息
- 目标职位：{{jobTitle}}
- 目标公司：{{company}}

## 你的任务
根据候选人的简历和目标职位，生成 4 种不同风格的求职话术。

## 话术要求

### 1. Boss 直聘话术（boss）
- 风格：亲切、简洁、直接
- 适合在线聊天，快速引起 HR 注意
- 突出核心优势和匹配度

### 2. 智联招聘话术（zhilian）
- 风格：专业、正式、有条理
- 适合投递简历时的自我介绍
- 强调技能匹配和项目经验

### 3. 牛客网话术（niuke）
- 风格：技术范、有深度
- 适合技术社区找内推
- 突出技术栈和项目亮点

### 4. 邮件投递话术（email）
- 风格：正式、礼貌、完整
- 适合邮件投递简历
- 包含完整的自我介绍和求职意向

## 输出格式
返回 JSON 格式，包含 4 个字段：
```json
{
  "boss": "Boss直聘话术内容",
  "zhilian": "智联招聘话术内容",
  "niuke": "牛客网话术内容",
  "email": "邮件投递话术内容"
}
```

每段话术控制在 100-200 字以内。用中文输出。
